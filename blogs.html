Question-1: What are the different ways to select an element in the DOM?
There are many different ways to select an element. Such as getElementById(), getElementsByClassName(),querySelector(). There are many more but these are used more frequently then others.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML = innerHTML creates html content.
innerText = innerText helps to set text.
textContent = textContent gets or sets the text content of an element.
Question-3: What is event delegation in the DOM?
When we add a single event listener to parent element instead of adding multiple event listener to the child element.
Question-4: What is event bubbling in the DOM?
When an event is triggered on a child element and it reaches to the parent element is called event bubbling.
Question-5: How do you create, add, and remove elements using JavaScript?
To create element we use document.createElement("div");
to add Something we use appendChild('div');
and to remove something we use element.remove;
